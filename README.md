## How to Run
1. Download Java JDK.
2. Clone the repository.

Option 1:
1. Open the terminal in Soen-342/src.
2. Run `javac Main.java`.
3. Run `java Main`.

Option 2:
1. Open Soen-342 on the IDE of your choice.
2. Navigate to src/Main.java.
3. Click Run.
