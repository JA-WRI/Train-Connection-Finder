## How to Run
1. Download Java JDK.
2. Download SQLite
3. Clone the repository.

Option 1:
1. Open the terminal in Soen-342/Backend/src.
2. Run `javac Main.java`.
3. Run `java Main`.
4. Open SOEN-342/Frontend/railway-frontend.
5. Run `npm start`.

Option 2:
1. Open Soen-342 on the IDE of your choice.
2. Navigate to src/com.trainapp.Main.java.
3. Click Run.
4. Open Soen-342 on the IDE of your choice.
5. Open SOEN-342/Frontend/railway-frontend.
6. Run `npm start`.
