## How to Run
1. Download Java JDK.
2. Download SQLite
3. Clone the repository.
4.  Open Soen-342 on the IDE of your choice.
5. Navigate to src/com.trainapp.Main.java.
6. Click Run.
7. Open Soen-342 on the IDE of your choice.
8. Open SOEN-342/Frontend/railway-frontend.
9. Run `npm i`
10. Run `npm start`.
